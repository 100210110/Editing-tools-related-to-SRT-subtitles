# Editing-tools-related-to-SRT-subtitles
跳过做字幕重复工作的小工具

## 启动工具箱主程序
- 启动后的大白框是拖动文件进去的，拖完后可以自己点选移除
- 会自动识别插件目录，存在对应的插件配置文件将会添加对应按钮
- 拖入文件后可以点击那些插件按钮来处理了

## Srt加前缀使用方法
- 做成现在这种集中管理后，没法直接拖文件上去处理了
- 点击加前缀按钮，会把输入文件前面加好说话人前缀，**使用的是srt的文件名**加中文冒号
- 处理后文件会暂存在缓存中，需继续处理可点其他插件，需要最终文件可导出

## Srt常见脏话和错误词矫正
- 做成现在这种集中管理后，没法直接拖文件上去处理了
- config.json为删改词的配置，可以自己修改来适配，没有的话会创建个默认的，格式自己看吧，很直观
- 修改config.json时小心格式错误，程序读取失败会直接覆盖
- delete中, **profanity**键对应列表的词会被**文明小助手计数**, **others**键对应列表的词则**仅删除不计数**
- 为了直观展示不同人的素质高低，本工具贴心附带了**文明小助手**功能，会在转换后按被删除的屏蔽词次数排名，助您更有针对性的提醒或更准确的背后**吐槽**。
- 处理后文件会暂存在缓存中，需继续处理可点其他插件，需要最终文件可导出
