# Editing-tools-related-to-SRT-subtitles
跳过做字幕重复工作的小工具

## Srt加前缀使用方法
- 把srt文件直接拖上去，会询问是否使用文件名作为讲话人前缀，无需输入冒号，程序自动加
- 多个srt拖上去，现在逻辑是：选y全都用文件名，选n每个文件都询问你自定义
- 哦，这个会**直接覆盖原文件**，没写放在文件夹，如果你需要保留原srt做副本，代码在这可以自己改

## Srt常见脏话和错误词矫正
- 也是把srt文件直接拖上去，同上
- config.json为删改词的配置，可以自己修改来适配，没有的话双击运行会创建个默认的，格式自己看吧，很直观
- 修改json时小心格式错误，程序读取失败会询问覆盖，不会直接覆盖
- 输出在output文件夹，别问上面那个怎么没写，当时懒的加，到这复制过来就顺便加了¯\_(ツ)_/¯
- 为了直观展示不同人的素质高低，本工具贴心附带了文明小助手功能，会在转换后按被删除的屏蔽词次数排名，助您更有针对性的提醒或更准确的背后吐槽。
